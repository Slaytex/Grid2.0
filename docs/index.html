Overview

Grid 2.0 is a powerful combination of a desktop application and Figma plugin that streamlines the process of creating and managing grid layouts in Figma. The system provides precise control over frame sizes and supports both custom dimensions and preset screen sizes.

Getting Started

  1. Download and install the Grid 2.0 desktop application
  2. Install the Grid 2.0 Figma plugin from the Figma plugin marketplace
  3. Launch the Grid App first - it will start a local server and wait for connections
  4. Open your Figma project and access the Grid 2.0 plugin

Frame Naming Conventions

Grid 2.0 supports two ways to specify frame sizes:

1. Custom Size Format

Use this format to create frames with custom dimensions:

frame-name#width×height

Example: MyFrame#10×5 creates a frame named "MyFrame" with width=10 inches and height=5 inches

2. Preset Size Format

Use this format to create frames with predefined sizes:

frame-name#code#

Example: MyFrame#SA# creates a frame with the preset "Standard A" size

Using the Plugin

  1. Make sure the Grid App is running and showing "Waiting for connection"
  2. Name your frames using either the custom size format or preset size codes
  3. Select the frames you want to process in Figma
  4. Open the Grid 2.0 plugin
  5. Click the refresh icon to update the frame list
  6. Click "Send" to process the selected frames
  7. The Grid App will automatically adjust the frame sizes according to your specifications

Troubleshooting

Common Issues

  • Ensure Grid App is running before using the Figma plugin
  • Verify frame names follow the correct format (either custom size or preset code)
  • Check if your firewall is blocking the connection between the app and plugin
  • Make sure frames are selected before clicking the send button
